# Node.js + Expressjs + Mongodb

## Requirements

- **nodejs >= v14.7.0**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn install
$ yarn start

- nodejs = v18.16.0
- npm = v9.8.1
- yarn = v1.22.19

```
